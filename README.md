# FunPay Hunter для Black Russia

Бот для отслеживания новых предложений Black Russia на FunPay.

## Особенности
- ✅ Отслеживает только онлайн продавцов
- ✅ Фильтрует по ключевым словам "Black Russia"
- ✅ Проверяет цену (10-50000 руб)
- ✅ Отправляет уведомления в Telegram
- ✅ Работает 24/7 на Render
- ✅ Веб-интерфейс для управления

## Установка

### 1. Подготовка
1. Создайте бота через @BotFather в Telegram
2. Получите токен бота (например: `1234567890:ABCdefGHIjklMnOpQRstUVwxyz`)
3. Узнайте свой Chat ID (спросите у @userinfobot)
4. Создайте аккаунт на [Render.com](https://render.com)

### 2. Развертывание
1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы из этого архива
3. На Render создайте новый "Web Service"
4. Подключите ваш GitHub репозиторий
5. Укажите настройки:
   - **Runtime:** Python 3
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
6. Добавьте переменные окружения:
   - `TELEGRAM_BOT_TOKEN` = токен вашего бота
   - `TELEGRAM_CHAT_ID` = ваш chat ID
7. Нажмите "Create Web Service"

### 3. Настройка вебхука (опционально)
1. После деплоя получите URL вашего сервиса
2. Установите вебхук: `https://api.telegram.org/bot{ВАШ_ТОКЕН}/setWebhook?url={URL_ВАШЕГО_СЕРВИСА}/webhook`

## Использование

### Веб-интерфейс
Откройте URL вашего сервиса:
- `/` - главная страница
- `/test` - тест парсинга
- `/start_monitor` - запуск мониторинга
- `/stop_monitor` - остановка мониторинга
- `/check` - ручная проверка
- `/status` - статус системы

### Команды Telegram
- `/start` - начать работу
- `/check` - проверить сейчас
- `/monitor` - запустить мониторинг
- `/stop` - остановить мониторинг
- `/status` - статус системы
- `/help` - помощь

## Логирование
Логи доступны в Render Dashboard во вкладке "Logs".

## Устранение неполадок
1. **Бот не отправляет сообщения:** проверьте TELEGRAM_BOT_TOKEN и TELEGRAM_CHAT_ID
2. **Парсинг не работает:** проверьте логи, возможно изменилась структура FunPay
3. **Сервис не запускается:** проверьте requirements.txt и логи сборки
4. **Нет уведомлений:** возможно нет онлайн продавцов в данный момент
